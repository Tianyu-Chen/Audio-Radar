About This Project

This project analyzes music trends and preferences across different countries using data from Last.fm and Spotify. We examine various audio features, provide personalized recommendations, and showcase popular tracks from around the world.

Key Features:

  • Interactive visualization of audio features by country
  • Personalized music recommendations based on audio preferences
  • Analysis of top songs across different regions
  • Comparison of musical characteristics between countries
